Generate model climate data

This module generates the climate forcing data. Climate forcing data are read from daily geotiff files and zonal means are computed for each subcatchment of the model. The resulting time series are stored in SWAT format. The climate folders containing daily geotiff files can be updated using OSFWF tools 1a-1c.

Get new NOAA-GFS data

Downloads the NCEP GFS Forecasts (.5 degree grid) data for issue hour 0 and forecast hours 6, 12, 18 and 24.
After download the files are converted to GeoTIFF files and computed into daily maps.

Data are downloaded from the NOAA Operational Model Archive and Distribution System.

Select precipitation folder

Folder where precipitation files will be saved. Only missing files and new forecasts will be downloaded (max. ~30 days). If none or non-existing folder is given, no files will be downloaded.

Select maximum temperature folder

Folder where maximum temperature files will be saved. Only missing files and new forecasts will be downloaded (max. ~30 days). If none or non-existing folder is given, no files will be downloaded.

Select minimum temperature folder

Folder where minimum temperature files will be saved. Only missing files and new forecasts will be downloaded (max. ~30 days). If none or non-existing folder is given, no files will be downloaded.

Advanced parameters

Advanced parameters are the boundary coordinates of the grid to be downloaded.

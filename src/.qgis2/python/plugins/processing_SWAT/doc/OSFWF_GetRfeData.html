Get FEWS-RFE data

Downloads the NOAA Daily RFE 2.0 Data of the Africa continental window.
After download the files are converted to GeoTIFF files.

Data will be processed one year at a time and downloaded from the FEWS NET Data portal (http://earlywarning.usgs.gov/fews/africa/web/dwndailyrfe.php).

Select precipitation folder

Folder where files will be saved to.

Start date

Format must be YYYYMMDD, e.g. 20120131. Data is available from 20010101.

End date

Same format as start date. No errors will be given for missing data.

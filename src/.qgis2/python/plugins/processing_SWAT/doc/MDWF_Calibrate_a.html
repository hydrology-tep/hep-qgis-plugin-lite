Generate PEST template files

Template files tell PEST where to find calibration parameters in the SWAT input files. This routine supports writing template files for all SWAT parameters. The routine can be run repeatedly and calibration parameters will be accumulated. If one single parameter should be calibrated for several basins, reaches or HRU's, simply re-use the parameter name.

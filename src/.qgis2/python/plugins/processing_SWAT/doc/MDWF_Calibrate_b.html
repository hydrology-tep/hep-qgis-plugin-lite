Generate PEST instruction file

Instruction files tell PEST where to find simulated observations. At the moment, only river discharge observations are supported by the module.

Select model source folder

Folder where instruction file will be saved to.

Select model description file

File with observation data (time, obs, error, reach ID)

River discharge observations need to be provided as simple .csv files with the date in the first column, the discharge (m3/s) in the second column, the error of the observation in the third column and corresponding reach ID in the fourth column. Dates should be EXCEL dates converted to numbers.

PEST name of observation group

Name of observation group which the observed data should be long to.

Temporal resolution of reach output and observation data

Choose the temporal resolution corresponding to the SWAT reach output file and observation data. The SWAT reach output file can have either daily or monthly time steps. If using monthly time steps the data of the observation data should be the 1st of every month.

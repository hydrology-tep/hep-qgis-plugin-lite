Select assimilation folder

Folder containing assimilation results.

Issue date (yyyy-mm-dd)

The issue date denotes the day for start of forecast and should be given in the following format: yyyy-mm-dd.

Select file with corresponding observations (time, obs, measurement error, reach ID)

River discharge observations need to be provided as simple .csv files with the date in the first column, the discharge (m3/s) in the second column, measurement error (m3/s) in the third column and reach ID in the fourth column.
If discharge data for multiple stations is used data should be placed below each other yielding a N x 4 matrix with N being the total number og observations.
Dates should be EXCEL dates converted to numbers.

Reach ID

Reach ID is utilised only, if observation file does not exist.

Generate SENSAN parameter variation file

Select model source folder

Folder where parameter variation file will be saved to.

Select source for parameter variation

Source for which the baseline parameter set should be based on for the parameter variation.
Deviation in parameter values for the additional parameter sets will be based on the baseline parameters.
If you choose the source to be the .pbf's no additional files should be selected as the programme automatically will identify the files in the model source folder.

Select PEST control file

If source for parameter variation is chosen as .pst, please select the PEST control file which the baseline parameter set should be based on.

Select PEST output parameter file

If source for parameter variation is chosen as .par, please select the PEST output parameter file which the baseline parameter set should be based on.

Percent deviation in parameter values

Percent in- or decrease in parameter value used in the sensitivity analysis.
Percent decrease should be given as a negative value e.g 5% decrease should be given as -5.

Generate Model Description and climate files

This module writes the model description file and empty SWAT climate files for the model. You need to specify the model subcatchment shapefile in lat-long and indicate which column of its attribute table holds the subcatchment IDs. You also need to specify the centroid shapefile and indicate columns for latitude, longitude and elevation.
